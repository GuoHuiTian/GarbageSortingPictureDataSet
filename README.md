# 垃圾分类图片数据集（120类）
## GarbageSortingPictureDataSet
项目说明：根据国内现有垃圾分类标准使用网络爬虫的方式进行数据采集，采集的目标URL为百度图片。
数据集包含可回收物、厨余垃圾、其他垃圾和有害垃圾四个大类，其他垃圾_PE塑料袋、其他垃圾_一次性棉签、其他垃圾_唱片、其他垃圾_打火机、其他垃圾_滚筒纸等120个小类共计39348张，可用于不同的深度学习模型训练。数据各类别的图片数量如图
